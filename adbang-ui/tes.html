Input Data Anggaran

Input data anggaran dipergunakan untuk menginput semua data anggaran realisasi per bulan. Harap untuk di isi segera sebelum waktu penginputan di tutup kembali.

  • Data yang di input berupa nominal saja yaitu anggaran realisasi.
  • Penginputan dilakukan secara bertahap di setiap bulan nya.
  • Data yang di input yaitu realisasi pada bulan yang dipilih.

Input Data Kinerja (K)

Input data kinerja dipergunakan untuk menginput semua data kinerja target dan realisasi kinerja per bulan. Harap untuk di isi segera sebelum waktu penginputan di tutup kembali.

  • Data yang di input dapat berupa nominal bilangan bulat ataupun desimal.
  • Penginputan dilakukan secara bertahap di setiap bulan nya.
  • Data yang di input yaitu realisasi pada bulan yang dipilih.
